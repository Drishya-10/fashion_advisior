Fashion AI Advisor

Fashion AI Advisor is an AI-powered fashion recommendation system that helps users discover similar clothing items, generate outfit suggestions, and receive fashion advice through a chatbot.

Features:
Upload a clothing image and find visually similar products.
Detect fashion items using YOLOv8.
Generate image embeddings with OpenCLIP.
Create complete outfit recommendations.
Chat with an AI fashion assistant powered by Groq LLM.
Personalized recommendations using user feedback.

How It Works
User uploads an image.
YOLO detects the clothing item.
OpenCLIP generates image embeddings.
Similar products are retrieved using cosine similarity.
Outfit recommendations are generated.
The AI chatbot provides styling suggestions.
